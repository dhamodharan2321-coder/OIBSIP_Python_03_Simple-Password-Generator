# 🔐 Dhamodharan Password Generator

An advanced **Random Password Generator** built using Python and Tkinter.  
This application allows users to generate strong and secure passwords with customizable options.

---

## 🚀 Features

✅ User-friendly GUI using Tkinter  
✅ Generate random secure passwords  
✅ Customize password length  
✅ Include/exclude:
- Letters (A–Z, a–z)
- Numbers (0–9)
- Symbols (!@#$...)  

✅ Strong password rules (ensures at least one selected type) 🔐  
✅ Removes confusing characters (O, 0, l, 1)  
✅ Copy password to clipboard 📋  
✅ Clear input fields 🧹  
✅ Error handling and input validation  

---

## 🛠 Technologies Used

- Python 🐍  
- Tkinter (GUI Development)  
- Random Module (Password generation)  
- String Module (Character sets)

## 🎯 How It Works

1. Enter desired password length  
2. Select character options (letters, numbers, symbols)  
3. Click **Generate Password**  
4. Copy the password using the **Copy button**
